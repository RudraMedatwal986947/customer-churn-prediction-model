# Skills & Technologies Acquired

Through the architecture, design, and ongoing implementation of the **Intelligent Customer Lifetime Value & Churn Prediction Platform**, the following skills, tools, and methodologies have been acquired and practically applied:

## 1. Machine Learning & Data Science
* **Predictive Modeling:** Designing algorithms to predict binary outcomes (Customer Churn) and continuous values (Customer Lifetime Value).
* **Customer Segmentation:** Applying RFM (Recency, Frequency, Monetary) analysis and K-Means clustering to identify targetable customer profiles.
* **Data Engineering & Preprocessing:** Structuring data pipelines to handle missing values, encode categorical features, and scale numeric data for model consumption.
* **Libraries:** `scikit-learn`, `xgboost`, `pandas`, `numpy`.

## 2. Backend & API Development
* **RESTful APIs:** Designing and building API endpoints to serve machine learning models and data insights to external clients.
* **Framework:** Leveraging **FastAPI** for high-performance, async-capable Python backend development.
* **Architecture:** Organizing routes, models, and application states cleanly in a modular format.

## 3. Database Management & ORM
* **Relational Databases:** Setting up and managing **PostgreSQL** to store complex structured data (demographics, financials, model predictions).
* **Object-Relational Mapping (ORM):** Using **SQLAlchemy** to abstract SQL queries and programmatically define database schemas using Python classes.

## 4. Frontend & Data Visualization
* **Rapid Prototyping:** Utilizing **Streamlit** to build an interactive web application entirely in Python without needing HTML/JS.
* **Data Storytelling:** Creating dedicated interfaces for Exploratory Data Analysis (EDA), segmentation insights, and live model predictions.

## 5. DevOps & Containerization
* **Docker:** Writing `Dockerfile`s to create isolated, reproducible environments for both the backend (FastAPI) and frontend (Streamlit) services.
* **Docker Compose:** Managing a multi-container application stack (Database + API + Dashboard) with a single `docker-compose.yml` configuration.
* **Environment Management:** Resolving dependencies and utilizing `requirements.txt` effectively.

## 6. Software Architecture & Project Management
* **System Design:** Architecting an end-to-end machine learning system connecting raw data ingestion to user-facing dashboards.
* **Project Structuring:** Establishing a maintainable and scalable directory layout separating concerns (data, models, api, frontend, database).
* **Technical Documentation:** Creating structured markdown documentation (Scopes, Implementation Plans, Walkthroughs) to track progress and align objectives.
